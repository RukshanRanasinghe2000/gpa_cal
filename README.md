# Getting Started to dev environment

Prerequisites
- Flutter (Latest version)
- Dart SDK
- Git
# Installation  
1. Clone the repo
    https://github.com/RukshanRanasinghe2000/gpa_cal.git
2. Install dependencies
   ```flutter pub get```
3. Run the app
   ```flutter run```
